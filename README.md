# Accounting Document Management System

This project is a web-based platform designed for accounting firms and their clients to manage, store, and track accounting documents securely and efficiently. The system helps reduce the risk of document loss, ensures legal compliance, and streamlines workflows between clients and accountants.

## Features
- Secure login with JWT and role-based access (CLIENT / ACCOUNTANT)
- Upload and manage accounting documents (PDF, JPG, PNG)
- Metadata tracking for each document (number, type, date, amount, supplier)
- Document approval and rejection workflow for accountants
- View documents by fiscal year with status updates
- Local secure file storage with automatic cleanup
- Quick and easy document search

## Technologies
- Backend: Spring Boot (REST API)
- Security: Spring Security, JWT
- Database: H2 (in-memory)
- Testing: JUnit, Mockito

## Purpose
The system provides a reliable solution for managing accounting documents, improving collaboration between clients and accountants, and ensuring compliance with legal retention requirements.
